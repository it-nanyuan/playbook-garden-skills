# Claude

## Purpose

Claude 目前更适合消费一份直接可读的指令文本，因此本项目使用 `agents/claude.md` 作为适配文件。

## Consumed File

- `agents/claude.md`

## Recommended Use

- 将这份文件作为 Claude Project instructions、system prompt 补充材料，或团队内部的共享指令模板
- 与仓库中的 `SKILL.md` 配合使用：`claude.md` 提供平台入口，`SKILL.md` 提供完整规范

## Maintenance Notes

- 保持内容简洁，突出输出格式、前缀约束和提交前先看 diff 的流程
- 如果后续 Claude 提供稳定的结构化 skill 配置格式，再考虑新增机器可读文件
