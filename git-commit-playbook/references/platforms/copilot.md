# GitHub Copilot

## Purpose

GitHub Copilot 可以通过项目说明或自定义指令消费文本规则，因此这里提供 `agents/copilot-instructions.md`。

## Consumed File

- `agents/copilot-instructions.md`

## Recommended Use

- 将该文件内容接入 Copilot instructions、仓库级开发规范，或作为辅助提示模板
- 当团队主要在 GitHub 流程里使用 AI 起草提交说明时，这份文件可以作为标准入口

## Maintenance Notes

- 内容应保持短而硬，强调输出格式和前缀强约束
- 如果后续出现官方支持的结构化仓库级 instruction 格式，再升级为对应产物
