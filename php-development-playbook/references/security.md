# PHP Security Standard

## Core Areas

- 参数校验
- 认证鉴权
- 敏感信息保护
- 上传下载安全
- 回调验签
- SQL 注入、XSS、CSRF、反序列化风险控制

## Rules

- 优先使用框架成熟安全能力，不重复造轮子
- 不信任任何外部输入，包括后台管理员输入和第三方回调
- 敏感字段脱敏后再写日志
- 文件上传要校验类型、大小、存储路径和访问控制
- 后台接口、导出接口、批量操作接口要特别关注权限范围

## High-Risk Paths

- 支付通知
- 第三方平台回调
- 文件上传与下载
- 管理后台高权限操作
- 批量修改与批量删除
