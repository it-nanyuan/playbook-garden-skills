# Libraries

## Common Choices

- UI：`Jetpack Compose`
- DI：`Hilt`
- Networking：`Retrofit`、`OkHttp`
- Serialization：`kotlinx.serialization`、`Moshi`
- Images：`Coil`
- Database：`Room`
- Async：`Coroutines`
- Paging：`Paging 3`
- Logging：`Timber`

## Rule

- 优先与现有项目保持一致
- 不要在同一个项目里引入过多重叠依赖
- 依赖选择先看稳定性、维护活跃度和团队熟悉度
