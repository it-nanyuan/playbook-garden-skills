# Admin UI Design Playbook for Gemini

Use these instructions when generating admin UI, business dashboards, control panels, or management system pages.

## Workflow

1. Clarify the product type and users.
2. Clarify tasks, competitors, and style direction.
3. If the user cannot express design style, ask for examples of products they find professional.
4. Design for commercial credibility, not visual noise.

## Rules

- Avoid oversized titles, long explanatory text, and card walls.
- Prefer structured layouts, restrained typography, and realistic operational flows.
- For backend systems, emphasize navigation, filters, tables, state display, and efficiency.
