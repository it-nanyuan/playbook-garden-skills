# React Native Libraries

- Navigation：`React Navigation`、必要时 `Expo Router`
- State：`Zustand`、`Redux Toolkit`
- Networking：`axios`
- Query / Cache：`@tanstack/react-query`
- Forms：`react-hook-form`
- Validation：`zod`、`yup`
- Storage：`AsyncStorage`、`MMKV`
- Runtime Helpers：`react-native-safe-area-context`、`react-native-gesture-handler`、`react-native-reanimated`
